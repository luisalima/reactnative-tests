# Run

## iOS

1. `$ open ios/WeatherProject.xcodeproj`
2. On xcode, hit `Build and run` button
3. Go to the `Simulator` and under `Debug > Location` select `Apple`
4. <kbd>cmd</kbd> + <kbd>r</kbd>
